1. Tính Đóng Gói (Encapsulation) & C# Properties

Thử nghiệm cơ chế bảo vệ dữ liệu nội bộ thông qua thuộc tính { get; set; }

Biến ẩn nội bộ (_age) 0
Property Công Khai (Age) 0
// Mã nguồn C# Đóng Gói thuộc tính đầy đủ (Full Property) public class Account
{
    private int _age; // Backing field (Bị giấu kín)

    public int Age
    {
        get => _age;
        set
        {
            if (value < 0) throw new ArgumentException("Lỗi tuổi âm!");
            _age = value;
        }
    }
}
💡 Góc nhìn FE: Giống như bạn viết một `set` trap trong ES6 Proxy hoặc getter/setter trong Class của TypeScript. Nó bảo vệ vùng nhớ nội bộ không bị ghi đè dữ liệu rác.
